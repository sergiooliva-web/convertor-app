# Image & Video Converter

A powerful web-based tool for converting images to WEBP and compressing videos with high quality. Built with Next.js 14 and FFmpeg.

## Features

- **Image Converter**: Convert PNG, GIF, JPEG to WEBP with batch processing
- **Video Compressor**: Compress videos without losing quality
- **Batch Processing**: Convert multiple files simultaneously
- **Real-time Progress**: Monitor conversion progress in real-time
- **Modern UI**: Clean, dark-themed interface with intuitive controls
- **Customizable**: Adjust quality, preset, and resolution settings
- **CPU/GPU Switching**: Toggle between CPU and GPU for faster processing
