# Changelog

## [1.0.0] - 09.07.2026

### Added
- Image converter (PNG, GIF, JPEG → WEBP)
- Video compressor with FFmpeg
- Batch processing for images and videos
- Real-time progress bar
- Dark theme UI
- Download individual files or all at once
- CRF quality control (18-28)
- Preset selection (Slow, Medium, Fast)
- Resolution scaling (1080p, 720p, 480p, 360p)
- Roboto font integration

### Fixed
- Multiple file download issue
- FFmpeg path detection
- Progress bar accuracy
